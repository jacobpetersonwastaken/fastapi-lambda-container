# FastAPI AWS Lambda Container
This is a simple api that is Docker containerized FastAPI Lambda function that accepts a query string parameter and returns a string.

## Tech used
- Docker
- Aws ECR(Elastic Container Registry)
- Aws Lambda
- FastAPI
